Investigation Overview¶

During the cause of investigating this dataset, features that increase the probability of being transported to a passeners destination was my main focus.This study can be used for predictive analysis upon further analysis. My main focus was on Age, CryoSleep,Homeplanet and Destination variables

Dataset Overview¶

This data set contains 8,693 rows with each row being an information about each passengers and 14 variables which describes the passengers. The dataset contains features like PassengerId, HomePlanet, Cabin, Destination, Transported, and many others. During the cleaning stage, I dropped a few columns which I felt would not in any way affect the analysis. Features like: PasssengerId and Name of Passenger
